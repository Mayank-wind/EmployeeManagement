# Employee Management System

A Full-Stack Employee Management System built using Spring Boot, MySQL, JPA, and REST APIs.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST API

## 📌 Features
- Add Employee
- Update Employee
- Delete Employee
- View All Employees
- View Employee By ID

## 🛠️ Database Setup
1. Create database:
   CREATE DATABASE employee;

2. Update application.properties with your MySQL username and password.

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/Mayank-wind/EmployeeManagement.git

2. Navigate into project folder:
   cd EmployeeManagement

3. Run the project:
   mvn spring-boot:run

Server will start at:
http://localhost:8080
